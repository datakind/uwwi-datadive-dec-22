2020 Zip+4 ADI Files v 3.2
These files contain a linkage between the Census block group level ADI score and the zip+4 centroids contained within each block group. The file contains six relevant fields:

TYPE: P indicates post office; U indicates unique zipcode; standard zip codes are left blank
ZIP_4: The 9 digit zip code ID 
FIPS: The block group Census ID
GISJOIN: Key linkage field to the block group shapefile served by NHGIS
ADI_NATRANK: National percentile of block group ADI score
ADI_STATERNK: State-specific decile of block group ADI score

DO NOT USE EXCEL TO OPEN THE FILES! - Excel has a record limit of roughly 1.4 million and does not warn you if you exceed the limit. Truncation of the data may occur!
